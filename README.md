## 📖 About

MyPomodoro Timer is a desktop Pomodoro timer for Windows with fully customizable work and break intervals. Most existing apps either have fixed intervals or charge money for customization — this project solves that problem.

---

## 🛠️ Tech Stack

| Category | Technologies |
|----------|-------------|
| 💻 Language / Platform | C#, .NET, WinForms |
| 🔧 Tools | Git, GitHub, Visual Studio |

---

## ✨ Features

- ⏱️ **Custom work interval** — set any duration you want
- ☕ **Custom short break** — flexible short break timer
- 🌿 **Custom long break** — flexible long break timer
- 🔔 **Notifications** — alert when interval ends
- ▶️ **Full playback controls** — start, pause, reset
- 💾 **Settings persistence** — your intervals are saved between sessions

## 🍅 What is Pomodoro Technique?

The Pomodoro Technique is a time management method:

| Step | Duration | Description |
|------|----------|-------------|
| 🎯 Work | 25 min (customizable) | Focused work session |
| ☕ Short break | 5 min (customizable) | Quick rest |
| 🌿 Long break | 15 min (customizable) | After every 4 sessions |
